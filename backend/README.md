KAI Backend - README
====================

This is a minimal Node/Express backend for the KAI starter project.
It includes endpoints to create Razorpay orders, verify payments and a webhook stub.

Quick start (local):
1. cd backend
2. cp .env.example .env  -> fill keys
3. npm install
4. npm run dev

Deploy:
- You can deploy to Render, Railway, Heroku or Cloud Run.
- Set environment variables in your service dashboard.
