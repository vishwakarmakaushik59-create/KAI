KAI Frontend - README
=====================

This is a Vite + React starter for KAI. It includes:
- ChatWindow (demo)
- RazorpayCheckout component (demo)
- Styles (dark glass look)
- Environment variables: VITE_BACKEND_URL, VITE_RAZORPAY_KEY_ID

Local dev:
1. cd frontend
2. npm install
3. npm run dev
